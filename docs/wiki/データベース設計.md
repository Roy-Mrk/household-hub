# データベース設計

## テーブル一覧

| テーブル名 | 概要 |
|---|---|
| `profiles` | ユーザープロフィール（表示名） |
| `income` | 収入エントリ |
| `expense` | 支出エントリ |
| `categories` | カテゴリマスター |
| `subcategories` | サブカテゴリマスター |
| `households` | 世帯 |
| `household_members` | 世帯メンバー |
| `household_invitations` | 世帯招待トークン |
| `settlements` | 精算レコード |
| `settlement_items` | 精算対象エントリ |

---

## profiles

| カラム | 型 | 制約 | 説明 |
|---|---|---|---|
| `user_id` | uuid | PK, FK → auth.users | Supabase Auth ユーザー ID |
| `display_name` | text | NOT NULL, 1〜50 文字 | 表示名 |
| `created_at` | timestamptz | NOT NULL, DEFAULT now() | 作成日時 |
| `updated_at` | timestamptz | NOT NULL | 更新日時（トリガーで自動更新） |

---

## income / expense

`income` と `expense` は同一の構造を持ちます。

| カラム | 型 | 制約 | 説明 |
|---|---|---|---|
| `id` | bigint | PK, AUTO_INCREMENT | エントリ ID |
| `user_id` | uuid | NOT NULL, FK → auth.users | 登録者 |
| `household_id` | uuid | FK → households | 所属世帯（NULL = 世帯なし） |
| `source` | text | NOT NULL, 1〜100 文字 | 収支元/用途（例:「給与」「食費」） |
| `amount` | bigint | NOT NULL, 0 以上, 最大 10 億 | 金額（円、小数なし） |
| `subcategory_id` | uuid | FK → subcategories | サブカテゴリ（省略時は「未分類」） |
| `entry_date` | date | NOT NULL | 発生日 |
| `owner` | text | NOT NULL, DEFAULT `'self'` | `'self'`（自分）/ `'shared'`（共有） |
| `needs_settlement` | boolean | NOT NULL, DEFAULT true | 精算未了フラグ |
| `created_at` | timestamptz | NOT NULL, DEFAULT now() | 作成日時 |
| `updated_at` | timestamptz | NOT NULL | 更新日時（トリガーで自動更新） |

---

## categories

| カラム | 型 | 制約 | 説明 |
|---|---|---|---|
| `id` | uuid | PK, DEFAULT gen_random_uuid() | カテゴリ ID |
| `name` | text | NOT NULL, 1〜30 文字 | カテゴリ名 |
| `type` | text | NOT NULL | `'income'` / `'expense'` |
| `user_id` | uuid | FK → auth.users, NULL 可 | NULL = マスターデータ（全ユーザー共通） |
| `sort_order` | int | NOT NULL, DEFAULT 0 | 表示順 |
| `created_at` | timestamptz | NOT NULL, DEFAULT now() | 作成日時 |

---

## subcategories

| カラム | 型 | 制約 | 説明 |
|---|---|---|---|
| `id` | uuid | PK, DEFAULT gen_random_uuid() | サブカテゴリ ID |
| `category_id` | uuid | NOT NULL, FK → categories | 親カテゴリ |
| `name` | text | NOT NULL, 1〜30 文字 | サブカテゴリ名 |
| `user_id` | uuid | FK → auth.users, NULL 可 | NULL = マスターデータ |
| `sort_order` | int | NOT NULL, DEFAULT 0 | 表示順 |
| `created_at` | timestamptz | NOT NULL, DEFAULT now() | 作成日時 |

---

## households

| カラム | 型 | 制約 | 説明 |
|---|---|---|---|
| `id` | uuid | PK, DEFAULT gen_random_uuid() | 世帯 ID |
| `name` | text | NOT NULL, 1〜50 文字 | 世帯名 |
| `created_by` | uuid | NOT NULL, FK → auth.users | 作成者（オーナー） |
| `created_at` | timestamptz | NOT NULL, DEFAULT now() | 作成日時 |

---

## household_members

| カラム | 型 | 制約 | 説明 |
|---|---|---|---|
| `household_id` | uuid | PK, FK → households | 世帯 ID |
| `user_id` | uuid | PK, UNIQUE, FK → auth.users | ユーザー ID（1 ユーザー = 1 世帯のみ） |
| `role` | text | NOT NULL | `'owner'` / `'member'` |
| `joined_at` | timestamptz | NOT NULL, DEFAULT now() | 参加日時 |

> **制約:** 1 ユーザーは同時に 1 世帯にしか所属できない（`user_id` に UNIQUE 制約）

---

## household_invitations

| カラム | 型 | 制約 | 説明 |
|---|---|---|---|
| `id` | uuid | PK, DEFAULT gen_random_uuid() | 招待トークン（UUID がそのままトークンとして機能） |
| `household_id` | uuid | NOT NULL, FK → households | 招待対象世帯 |
| `created_by` | uuid | NOT NULL, FK → auth.users | 招待作成者 |
| `expires_at` | timestamptz | NOT NULL | 有効期限（デフォルト 72 時間） |
| `used_at` | timestamptz | NULL 可 | 使用日時（NULL = 未使用） |
| `used_by` | uuid | FK → auth.users, NULL 可 | 使用ユーザー |
| `created_at` | timestamptz | NOT NULL, DEFAULT now() | 作成日時 |

---

## settlements

| カラム | 型 | 制約 | 説明 |
|---|---|---|---|
| `id` | uuid | PK, DEFAULT gen_random_uuid() | 精算 ID |
| `household_id` | uuid | NOT NULL, FK → households | 対象世帯 |
| `split_ratios` | jsonb | NOT NULL | 割り勘比率配列（下記参照） |
| `total_amount` | bigint | NOT NULL | 精算対象の純支出合計（支出 − 収入） |
| `payments` | jsonb | NOT NULL | 支払フロー配列（下記参照） |
| `settled_at` | timestamptz | NOT NULL, DEFAULT now() | 精算日時 |
| `cancelled_at` | timestamptz | NULL 可 | キャンセル日時（NULL = 有効） |
| `created_at` | timestamptz | NOT NULL, DEFAULT now() | 作成日時 |

**split_ratios の型:**
```json
[
  { "user_id": "uuid", "display_name": "田中", "ratio": 60 },
  { "user_id": "uuid", "display_name": "鈴木", "ratio": 40 }
]
```

**payments の型:**
```json
[
  {
    "from_user_id": "uuid",
    "from_name": "鈴木",
    "to_user_id": "uuid",
    "to_name": "田中",
    "amount": 3000
  }
]
```

## RLS ポリシー

### income / expense

| 操作 | 条件 |
|---|---|
| SELECT | `user_id = auth.uid()` OR `household_id` が自分の所属世帯 |
| INSERT | `user_id = auth.uid()` AND（`household_id` IS NULL OR 自分の所属世帯） |
| UPDATE | `user_id = auth.uid()` のみ |
| DELETE | `user_id = auth.uid()` のみ |

### categories / subcategories

| 操作 | 条件 |
|---|---|
| SELECT | `user_id IS NULL`（マスター）OR `user_id = auth.uid()` |
| INSERT | `user_id = auth.uid()` |
| UPDATE | `user_id = auth.uid()` |
| DELETE | `user_id = auth.uid()` |

### households

| 操作 | 条件 |
|---|---|
| SELECT | 自分が `household_members` に存在する世帯 |
| INSERT | `created_by = auth.uid()` |
| DELETE | `created_by = auth.uid()`（オーナーのみ） |

### household_members

| 操作 | 条件 |
|---|---|
| SELECT | 同じ世帯のメンバー |
| INSERT | `user_id = auth.uid()`（自分のみ参加可） |
| DELETE | `user_id = auth.uid()`（自分のみ退出可） |

### household_invitations

| 操作 | 条件 |
|---|---|
| SELECT | 認証済みユーザー全員（招待受諾のため） |
| INSERT | `created_by = auth.uid()` AND 自分がその世帯のオーナー |
| DELETE | `household_id` が自分の世帯 AND 自分がオーナー |

### settlements / settlement_items

| 操作 | 条件 |
|---|---|
| SELECT | `household_id` / `settlement_id` が自分の所属世帯 |
| INSERT | `household_id` / `settlement_id` が自分の所属世帯 |
| UPDATE | `household_id` が自分の所属世帯（キャンセル用） |
| DELETE | `household_id` / `settlement_id` が自分の所属世帯 |
