# household-hub Wiki

## 概要

household-hub は家計の収支（収入・支出）を管理するWebアプリです。世帯メンバーで共有・精算できます。

## プロジェクト概要

| 項目 | 内容 |
|------|------|
| UIの言語 | 日本語 |
| 認証 | メール/パスワード、Google OAuth |
| 共有 | 世帯単位でデータ共有 |
| 精算 | 共有支出・収入の割り勘計算 |
| テーマ | ダークテーマ |

## 技術スタック

| レイヤー | 技術 |
|----------|------|
| フレームワーク | Next.js 15 (App Router) |
| 言語 | TypeScript / JavaScript |
| スタイリング | Tailwind CSS |
| データベース | Supabase (PostgreSQL) |
| 認証 | Supabase Auth |
| バリデーション | Zod |
| グラフ | Recharts |
| ロギング | Winston |
