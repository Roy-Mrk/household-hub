# household-hub Wiki

household-hub は家庭の収支（収入・支出）を管理するウェブアプリケーションです。世帯メンバーで共有しながら精算まで行えます。

## プロジェクト概要

| 項目 | 内容 |
|---|---|
| UI 言語 | 日本語 |
| 認証 | メール/パスワード・Google OAuth |
| 共有機能 | 世帯（Household）単位でデータ共有 |
| 精算機能 | 共有支出・収入の割り勘計算 |
| テーマ | ダークテーマ |

## 技術スタック

| レイヤー | 技術 |
|---|---|
| フレームワーク | Next.js 15 (App Router) |
| 言語 | TypeScript / JavaScript |
| スタイリング | Tailwind CSS |
| データベース | Supabase (PostgreSQL) |
| 認証 | Supabase Auth |
| バリデーション | Zod |
| グラフ | Recharts |
| ロギング | Winston |
